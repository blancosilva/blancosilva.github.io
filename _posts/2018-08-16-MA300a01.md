 A. Write each of the following sets by listing their elements between braces.

1. \\( \\{ 5x-1 : x \in \mathbb{Z} \\} \\)
2. \\( \\{ 3x+2 : x \in \mathbb{Z} \\} \\)
3. \\( \\{ x \in \mathbb{Z} : -2 \leq x < 7\\} \\)

6. \\( \\{ x \in \mathbb{R} : x^2=9 \\} \\)
